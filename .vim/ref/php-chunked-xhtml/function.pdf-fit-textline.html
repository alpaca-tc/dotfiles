PDF_fit_textline

(PECL pdflib >= 2.0.0)

PDF_fit_textline — 1 行分のテキストを配置する

説明

bool PDF_fit_textline ( resource $pdfdoc , string $text , float $x , float $y , string $optlist )

1 行分のテキストを、さまざまなオプションに基づいてページ上に配置します。 成功した場合に TRUE を、失敗した場合に FALSE を返します。
