SoapClient::__call

(PHP 5 >= 5.0.1)

SoapClient::__call — SOAP 関数をコールする (非推奨)

説明

public mixed SoapClient::__call ( string $function_name , string $arguments )

このメソッドは非推奨です。かわりに SoapClient::__soapCall を使用してください。
