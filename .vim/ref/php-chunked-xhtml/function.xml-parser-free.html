xml_parser_free

(PHP 4, PHP 5)

xml_parser_free — XML パーサを解放する

説明

bool xml_parser_free ( resource $parser )

指定した XML パーサ parser を解放します。

パラメータ

parser
解放したい XML パーサへのリファレンス。

返り値

この関数は、parser が有効なパーサを 参照しない場合に FALSE、それ以外の場合にパーサを解放し、TRUE を返します。
