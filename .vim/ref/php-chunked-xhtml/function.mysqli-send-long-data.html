mysqli_send_long_data

(PHP 5 <= 5.4.0)

mysqli_send_long_data — mysqli_stmt_send_long_data() のエイリアス

説明

この関数は mysqli_stmt_send_long_data() のエイリアスです。

注意

注意:

mysqli_send_long_data() は非推奨で、廃止予定です。
