Imagick::flattenImages

(PECL imagick 2.0.0)

Imagick::flattenImages — 画像シーケンスをマージする

説明

Imagick Imagick::flattenImages ( void )

画像シーケンスをマージします。 これは、たとえば Photoshop のレイヤを単一の画像にまとめる場合などに便利です。

返り値

成功した場合に TRUE を返します。

エラー / 例外

エラー時に ImagickException をスローします。
