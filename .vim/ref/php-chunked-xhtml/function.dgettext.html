dgettext

(PHP 4, PHP 5)

dgettext — 現在のドメインを上書きする

説明

string dgettext ( string $domain , string $message )

dgettext() 関数により、単一のメッセージ参照について 現在のドメインを上書きすることができます。

パラメータ

domain

ドメイン。

message

メッセージ。

返り値

成功した場合に文字列を返します。
