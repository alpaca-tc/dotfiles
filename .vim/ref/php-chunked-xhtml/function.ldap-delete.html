ldap_delete

(PHP 4, PHP 5)

ldap_delete — ディレクトリからエントリを削除する

説明

bool ldap_delete ( resource $link_identifier , string $dn )

指定したエントリを LDAP ディレクトリから削除します。

パラメータ

link_identifier

ldap_connect() が返す LDAP リンク ID。

dn

LDAP エンティティの識別名。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
