Imagick::getImagePixelColor

(PECL imagick 2.0.0)

Imagick::getImagePixelColor — 指定したピクセルの色を返す

説明

ImagickPixel Imagick::getImagePixelColor ( int $x , int $y )

指定したピクセルの色を返します。

パラメータ

x

ピクセルの x 座標。

y

ピクセルの y 座標。

返り値

指定した座標の色を表す ImagickPixel のインスタンスを返します。

エラー / 例外

エラー時に ImagickException をスローします。
