SplDoublyLinkedList::offsetUnset

(PHP 5 >= 5.3.0)

SplDoublyLinkedList::offsetUnset — 指定した $index の値を削除する

説明

void SplDoublyLinkedList::offsetUnset ( mixed $index )

指定したインデックスの値を削除します。

パラメータ

index

削除したいインデックス。

返り値

値を返しません。

エラー / 例外

index が範囲外を指すときや index が整数として解釈できないときに OutOfRangeException をスローします。
