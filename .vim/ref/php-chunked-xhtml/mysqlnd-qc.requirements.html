要件

PHP 5.3.3 またはそれ以降の PHP 5.3。 PHP 5.4.0 は対応準備中です。

PECL/mysqlnd_qc は mysqlnd プラグインで、mysqlnd ライブラリに差し込まれます。 このプラグインを PHP の MySQL 拡張モジュール (mysqli, mysql, PDO_MYSQL) で使うには、その拡張モジュールで mysqlnd ライブラリを使うようにしなければなりません。 詳細は、それぞれの拡張モジュールのマニュアルを参照ください。

APC ストレージ・ハンドラを使用するには、 APC 3.1.3p1-beta またはそれ以降。

MEMCACHE ストレージ・ハンドラを使用するには、 libmemcache 0.38 またはそれ以降。

sqlite ストレージ・ハンドラを使用するには、 PHP にバンドルされた sqlite3。
