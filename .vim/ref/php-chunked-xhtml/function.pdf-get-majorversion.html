PDF_get_majorversion

(PHP 4 >= 4.2.0, PECL pdflib >= 1.0.0)

PDF_get_majorversion — メジャーバージョン番号を取得する [古い関数]

説明

int PDF_get_majorversion ( void )

この関数は PDFlib バージョン 5 で廃止されました。 かわりにパラメータ major を用いて PDF_get_value() を使用してください。
