OCI-Collection::free

(PHP 5, PECL OCI8 >= 1.1.0)

OCI-Collection::free — コレクションオブジェクトに関連付けられたリソースを解放する

説明

bool OCI-Collection::free ( void )

コレクションオブジェクトに関連付けられたリソースを解放します。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
