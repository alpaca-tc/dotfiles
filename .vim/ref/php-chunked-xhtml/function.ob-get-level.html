ob_get_level

(PHP 4 >= 4.2.0, PHP 5)

ob_get_level — 出力バッファリング機構のネストレベルを返す

説明

int ob_get_level ( void )

出力バッファリングハンドラのネストレベルを返します。

返り値

出力バッファリングハンドラのネストレベルを返します。 バッファリングがアクティブでない場合はゼロを返します。

参考

  • ob_start() - 出力のバッファリングを有効にする
  • ob_get_contents() - 出力用バッファの内容を返す
