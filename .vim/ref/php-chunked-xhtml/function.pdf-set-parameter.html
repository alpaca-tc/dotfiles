PDF_set_parameter

(PHP 4, PECL pdflib >= 1.0.0)

PDF_set_parameter — 文字列パラメータを設定する

説明

bool PDF_set_parameter ( resource $p , string $key , string $value )

PDFlib パラメータを文字列型で設定します。成功した場合に TRUE を、失敗した場合に FALSE を返します。
