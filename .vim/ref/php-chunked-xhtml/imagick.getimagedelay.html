Imagick::getImageDelay

(PECL imagick 2.0.0)

Imagick::getImageDelay — 画像の遅延を取得する

説明

int Imagick::getImageDelay ( void )

画像の遅延を取得します。

返り値

画像の遅延を返します。

エラー / 例外

エラー時に ImagickException をスローします。
