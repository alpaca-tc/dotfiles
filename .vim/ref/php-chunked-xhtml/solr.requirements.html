要件

PHP Ver 5.2.11 以降が必要です。

Apache Solr エクステンションが利用可能であるためには、 libxml および curl エクステンションも使用可能でなければいけません。

libxml2 2.6.31 以降が必要です。

libcurl 7.18.0 以降も必要です。

上記のライブラリのバージョンが必要です。コンパイルするためのコードをハックしようとすることは強く反対されます。 恐らく、デバックしにくくなる可能性のあるエラーで失敗するでしょう。
