基本的な VarnishLog の使用法

共用メモリから varnish ログラインを読み取る例を示します

例1 varnish 共用メモリ・ログを読み取り

<?php

$vl = new VarnishLog;
while(1) {
    $line = $vl->getLine();
    printf("%s %d %s", VarnishLog::getTagName($line['tag'], $line['id'],
    $line['data']);
}

exit(0);
?>
