実行時設定

php.ini の設定により動作が変化します。

cURL 設定オプション
名前 デフォルト 変更可能 変更履歴
curl.cainfo NULL PHP_INI_SYSTEM PHP 5.3.7 以降で利用可能
PHP_INI_* モードの詳細および定義については どこで設定を行うのか を参照してください。

以下に設定ディレクティブに関する 簡単な説明を示します。

curl.cainfo string

CURLOPT_CAINFO オプションのデフォルト値。 絶対パスで指定する必要があります。
