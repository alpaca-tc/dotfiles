mysqli_bind_result

(PHP 5 <= 5.4.0)

mysqli_bind_result — mysqli_stmt_bind_result() のエイリアス

説明

この関数は mysqli_stmt_bind_result() のエイリアスです。

注意

注意:

mysqli_bind_result() は非推奨で、廃止予定です。

参考

  • mysqli_stmt_bind_result() - 結果を保存するため、プリペアドステートメントに変数をバインドする
