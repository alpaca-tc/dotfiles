SWFShape::drawArc

(PHP 4 >= 4.0.5)

説明

void SWFShape::drawArc ( float $r , float $startAngle , float $endAngle )
警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

返り値

値を返しません。
