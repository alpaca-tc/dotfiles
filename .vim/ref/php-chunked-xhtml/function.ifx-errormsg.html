ifx_errormsg

(PHP 4, PHP <=5.2.0)

ifx_errormsg — 直近の Informix コールのエラーメッセージを返す

説明

string ifx_errormsg ([ int $errorcode ] )

直近の Informix エラーに関する Informix エラーメッセージを返します。

パラメータ

errorcode

指定すると、指定したコードに対応するエラーメッセージを返します。

返り値

エラーメッセージを文字列で返します。

例

例1 ifx_errormsg() の例

<?php
printf("%s\n<br>", ifx_errormsg(-201));
?>
