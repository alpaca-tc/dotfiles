Imagick::getVersion

(PECL imagick 2.0.0)

Imagick::getVersion — ImageMagick API のバージョンを返す

説明

array Imagick::getVersion ( void )

ImageMagick API のバージョンを文字列と数値で返します。

返り値

ImageMagick API のバージョンを文字列と数値で返します。

エラー / 例外

エラー時に ImagickException をスローします。
