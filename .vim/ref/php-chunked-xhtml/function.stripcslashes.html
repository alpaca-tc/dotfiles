stripcslashes

(PHP 4, PHP 5)

stripcslashes — addcslashes() でクォートされた文字列をアンクォートする

説明

string stripcslashes ( string $str )

バックスラッシュを取り除いた文字列を返します。C言語と同様に \n, \r ..., 8進表現, 16進表現を認識します。

パラメータ

str

元に戻したい文字列。

返り値

元に戻した文字列を返します。

参考

  • addcslashes() - C 言語と同様にスラッシュで文字列をクォートする
