MongoCursor::hint

(PECL mongo >=0.9.0)

MongoCursor::hint — クエリについてのヒントをデータベースに与える

説明

public MongoCursor MongoCursor::hint ( array $key_pattern )

パラメータ

key_pattern

クエリで使用するインデックス。

返り値

このカーソルを返します。

エラー / 例外

このカーソルの反復処理が始まっている場合に MongoCursorException をスローします。
