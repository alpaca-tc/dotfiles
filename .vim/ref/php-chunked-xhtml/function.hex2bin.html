hex2bin

(PHP >= 5.4.0)

hex2bin — 16進表現のデータを2進表現に変換する

説明

string hex2bin ( string $data )

16進表現のデータを2進表現に変換します。

パラメータ

data

16進表現のデータ。

返り値

指定したデータを2進表現に変換したものを返します。

例

例1 hex2bin() の例

<?php
$hex = hex2bin("6578616d706c65206865782064617461");
var_dump($hex);
?>

上の例の出力は、 たとえば以下のようになります。

string(16) "example hex data"
