Imagick::setImageIndex

(PECL imagick 2.0.0)

Imagick::setImageIndex — イテレータの位置を設定する

説明

bool Imagick::setImageIndex ( int $index )

画像リスト内でのイテレータの位置、index パラメータで指定した場所に設定します。

このメソッドは非推奨です。 Imagick::setIteratorIndex() を参照ください。

パラメータ

index

イテレータを設定する位置。

返り値

成功した場合に TRUE を返します。

エラー / 例外

エラー時に ImagickException をスローします。
