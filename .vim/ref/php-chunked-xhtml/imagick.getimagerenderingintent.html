Imagick::getImageRenderingIntent

(PECL imagick 2.0.0)

Imagick::getImageRenderingIntent — 画像のレンダリング方向を取得する

説明

int Imagick::getImageRenderingIntent ( void )

画像のレンダリング方向を取得します。

返り値

画像の レンダリング方向 を返します。

エラー / 例外

エラー時に ImagickException をスローします。
