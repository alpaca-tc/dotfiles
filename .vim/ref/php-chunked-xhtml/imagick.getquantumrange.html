Imagick::getQuantumRange

(PECL imagick 2.0.0)

Imagick::getQuantumRange — Imagick quantum range を返す

説明

array Imagick::getQuantumRange ( void )

Imagick quantum range を文字列で返します。

返り値

Imagick quantum range を文字列で返します。

エラー / 例外

エラー時に ImagickException をスローします。
