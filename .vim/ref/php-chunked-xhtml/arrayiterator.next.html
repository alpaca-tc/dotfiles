ArrayIterator::next

(PHP 5 >= 5.0.0)

ArrayIterator::next — 次のエントリに移動する

説明

public void ArrayIterator::next ( void )

この関数は次のエントリにイテレータを移動します。

パラメータ

この関数にはパラメータはありません。

返り値

値を返しません。

例

例1 ArrayIterator::next() の例

<?php
$arrayobject = new ArrayObject();

$arrayobject[] = 'zero';
$arrayobject[] = 'one';

$iterator = $arrayobject->getIterator();

while($iterator->valid()) {
    echo $iterator->key() . ' => ' . $iterator->current() . "\n";

    $iterator->next();
}
?>

上の例の出力は以下となります。

0 => zero
1 => one
