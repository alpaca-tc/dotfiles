DOMNode::hasChildNodes

(PHP 5)

DOMNode::hasChildNodes — ノードが子を保持しているかどうかを調べる

説明

public bool DOMNode::hasChildNodes ( void )

この関数は、ノードが子を保持しているかどうかを調べます。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
