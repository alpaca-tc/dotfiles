eio_busy

(PECL eio >= 0.0.1dev)

eio_busy — Artificially increase load. Could be useful in tests, benchmarking.

説明

resource eio_busy ( int $delay [, int $pri = EIO_PRI_DEFAULT [, callback $callback = NULL [, mixed $data = NULL ]]] )

eio_busy() artificially increases load taking delay seconds to execute. May be used for debugging, or benchmarking.

パラメータ

delay

Delay in seconds

pri

リクエストの優先順位。EIO_PRI_DEFAULT、EIO_PRI_MIN、EIO_PRI_MAX あるいは NULL を指定します。 NULL を渡した場合、pri は内部的に EIO_PRI_DEFAULT となります。

callback

This callback is called when all the group requests are done.

data

Arbitrary variable passed to callback.

返り値

eio_busy() returns request resource on success or FALSE on error.
