mysqli_get_metadata

(PHP 5 <= 5.4.0)

mysqli_get_metadata — mysqli_stmt_result_metadata() のエイリアス

説明

この関数は mysqli_stmt_result_metadata() のエイリアスです。

注意

注意:

mysqli_get_metadata() は非推奨で、廃止予定です。
