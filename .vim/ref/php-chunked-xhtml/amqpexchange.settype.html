AMQPExchange::setType

(PECL amqp >= Unknown)

AMQPExchange::setType — exchange の型を設定する

説明

public string AMQPExchange::setType ( string $exchange_type )

exchange の型を設定します。 AMQP_EX_TYPE_DIRECT, AMQP_EX_TYPE_FANOUT, AMQP_EX_TYPE_HEADER あるいは AMQP_EX_TYPE_TOPIC のいずれかとなります。

警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

exchange_type

exchange の型を文字列で指定します。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
