ming_setscale

(PHP 4 >= 4.0.5, PHP 5, PECL ming SVN)

ming_setscale — グローバルなスケール要素を設定する

説明

void ming_setscale ( float $scale )

出力する SWF の倍率を設定します。 SWF ファイル内では、座標は PIXEL ではなく TWIPS で計算します。 1 ピクセルは 20 TWIPS にあたります。

パラメータ

scale

設定する倍率。

返り値

値を返しません。
