ocifetchinto

(PHP 4, PHP 5, PECL OCI8 >= 1.0.0)

ocifetchinto — 結果配列の次の行を取得する (非推奨)

説明

int ocifetchinto ( resource $statement , array &$result [, int $mode = OCI_ASSOC + OCI_NUM ] )

この関数は推奨されません。代わりに推奨されるものは、 oci_fetch_array()、 oci_fetch_object()、 oci_fetch_assoc() および oci_fetch_row() です。

警告

この関数は、PHP 5.4.0 以降では非推奨となりました。 この関数は決して使わないようにしましょう。
