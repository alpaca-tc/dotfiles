Imagick::pingImage

(PECL imagick 2.0.0)

Imagick::pingImage — 画像の基本属性を取得する

説明

bool Imagick::pingImage ( string $filename )

このメソッドを使用すると、画像全体を読み込まなくても 画像の幅や高さ、サイズそしてフォーマットを取得できます。

パラメータ

filename

情報を読み込むファイル名。

返り値

成功した場合に TRUE を返します。
