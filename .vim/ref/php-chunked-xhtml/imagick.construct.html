Imagick::__construct

(PECL imagick 2.0.0)

Imagick::__construct — Imagick のコンストラクタ

説明

Imagick::__construct ( mixed $files )

Creates an Imagick instance for a specified image or set of images.

パラメータ

files

読み込みたい画像へのパス、あるいは複数のパスの配列。 パスのファイル名にはワイルドカードを含めることができ、 また URL を指定することもできます。

返り値

成功した場合に新しい Imagick オブジェクトを返します。

エラー / 例外

エラー時に ImagickException をスローします。
