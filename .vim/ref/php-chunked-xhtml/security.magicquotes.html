警告

この機能は PHP 5.3.0 で 非推奨となり、 PHP 5.4.0 で削除されました。

マジッククオートは、PHPスクリプトに入力されるデータを 自動的にエスケープする機能です。 コードでは、マジッククオートをオフにして 実行する際必要な時にデータをエスケープすることが望まれます。
