Imagick::getImageInterlaceScheme

(PECL imagick 2.0.0)

Imagick::getImageInterlaceScheme — 画像のインターレース手法を取得する

説明

int Imagick::getImageInterlaceScheme ( void )

画像のインターレース手法を取得します。

返り値

成功した場合に画像のインターレース手法を返します。 エラー時に ImagickException をスローします。
