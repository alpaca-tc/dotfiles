posix_getlogin

(PHP 4, PHP 5)

posix_getlogin — ログイン名を返す

説明

string posix_getlogin ( void )

現在のプロセスを所有するユーザのログイン名を返します。

返り値

ユーザのログイン名を文字列で返します。

例

例1 posix_getlogin() の使用例

<?php
echo posix_getlogin(); //apache
?>
