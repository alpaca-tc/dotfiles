MongoId::getPID

(PECL mongo >= 1.0.11)

MongoId::getPID — この id を作るために使うプロセス ID を取得する

説明

public int MongoId::getPID ( void )

パラメータ

この関数にはパラメータはありません。

返り値

この MongoId を作るために使う PID を返します。
