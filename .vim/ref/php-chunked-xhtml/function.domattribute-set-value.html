DomAttribute::set_value

(PHP 4 >= 4.1.0)

DomAttribute::set_value — 属性の値を設定する

説明

bool DomAttribute::set_value ( string $content )

この関数は属性の値を設定します。

パラメータ

content

新しい値

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。

PHP 5 への移行

DOMAttr の value プロパティを設定してください。
