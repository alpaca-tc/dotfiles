Transliterator::__construct

(PHP >= 5.4.0)

Transliterator::__construct — Transliterator オブジェクトを作成する

説明

Transliterator::__construct() ( void )

Transliterator オブジェクトを作成します。

警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

この関数にはパラメータはありません。

返り値

Transliterator オブジェクトを返します。

  • Transliterator::getErrorMessage() - 直近のエラーメッセージを取得する
  • Transliterator::create() - Transliterator を作成する
