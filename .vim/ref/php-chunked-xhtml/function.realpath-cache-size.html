realpath_cache_size

(PHP 5 >= 5.3.2)

realpath_cache_size — realpath キャッシュサイズを取得

説明

int realpath_cache_size ( void )

realpath キャッシュで使用されるメモリー量を取得します。

返り値

realpath キャッシュが使用中のメモリ量を返します。

例

例1 realpath_cache_size() 例

<?php
var_dump(realpath_cache_size());
?>

上の例の出力は、 たとえば以下のようになります。

int(412)
