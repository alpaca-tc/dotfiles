posix_getuid

(PHP 4, PHP 5)

posix_getuid — 現在のプロセスの実際のユーザ ID を返す

説明

int posix_getuid ( void )

現在のプロセスの実際のユーザ ID を返します。

返り値

ID を表す整数値を返します。

例

例1 posix_getuid() の使用例

<?php
echo posix_getuid(); //10000
?>
