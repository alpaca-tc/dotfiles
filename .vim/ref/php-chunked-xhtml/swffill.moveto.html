SWFFill::moveTo

(PHP 4 >= 4.0.5)

SWFFill::moveTo — 塗りつぶしの原点を移動する

説明

void SWFFill::moveTo ( float $x , float $y )
警告

この関数は、 実験的 なものです。この関数の動作・ 名前・その他ドキュメントに書かれている事項は、予告なく、将来的な PHP のリリースにおいて変更される可能性があります。 この関数は自己責任で使用してください。

塗りつぶしの原点を、指定したグローバル座標に移動します。

パラメータ

x

X 座標。

y

Y 座標。

返り値

値を返しません。
