OCI-Lob::eof

(PHP 5, PECL OCI8 >= 1.1.0)

OCI-Lob::eof — ラージオブジェクトのディスクリプタが EOF かどうかを調べる

説明

bool OCI-Lob::eof ( void )

ラージオブジェクトの内部ポインタが LOB の最後にあるかどうかを調べます。

返り値

もし、ラージオブジェクトの内部ポインタが LOB の最後にあるとき、 TRUE を返します。その他の場合は FALSE を返します。
