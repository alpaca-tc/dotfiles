RecursiveFilterIterator クラス

(PHP 5 >= 5.1.0)

導入

この抽象イテレータは、不要な値を RecursiveIterator から取り除きます。 このクラスを継承して、独自のフィルタを実装することができます。 RecursiveFilterIterator::accept() は、サブクラスで必ず実装しなければなりません。

クラス概要
