トラブルシューティング

何か問題が発生した場合は、これらが参考になるでしょう。

  • IRC

    MongoDB の公式 IRC ルームは irc.freenode.net/#mongodb です。ここが、一番すばやく反応が得られるところでしょう… 人がいれば。

  • メーリングリスト

    » MongoDB のメーリングリスト に質問すれば、(通常はとても早く反応があり) よい答えが得られるでしょう。

  • バグトラッカー

    バグを見つけた? 機能を追加して欲しい? 疑問がある? » PHP ドライバ用のバグトラッカー にそれを登録してください。

詳細なデバッグ出力を有効にするには、ドライバのコンパイル時にデバッグ用のフラグを指定します。 

$ ./configure CFLAGS=-DDEBUG

DEBUG は、すべてのデバッグ機能を有効にします。 すべてではなく、特定のデバッグフラグだけを指定することもできます。 最新のコードで使えるフラグは次のとおりです。

  • DEBUG_CONN

    接続のデバッグ。
