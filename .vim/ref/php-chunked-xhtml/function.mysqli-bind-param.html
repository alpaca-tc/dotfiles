mysqli_bind_param

(PHP 5 <= 5.4.0)

mysqli_bind_param — mysqli_stmt_bind_param() のエイリアス

説明

この関数は mysqli_stmt_bind_param() のエイリアスです。

注意

注意:

mysqli_bind_param() は非推奨で、将来は削除される予定です。
