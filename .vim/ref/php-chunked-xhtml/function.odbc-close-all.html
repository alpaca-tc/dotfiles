odbc_close_all

(PHP 4, PHP 5)

odbc_close_all — 全ての ODBC 接続を閉じる

説明

void odbc_close_all ( void )

odbc_close_all()は、 データベースサーバへの全ての接続を閉じます。

パラメータ

この関数にはパラメータはありません。

返り値

値を返しません。

注意

注意:

この関数は、ある接続においてオープンされたトランザクションがある場合に失敗します。 この場合、この接続はオープンされたままとなります。
