MongoDB::getSlaveOkay

(PECL mongo >=1.1.0)

MongoDB::getSlaveOkay — このデータベースの slaveOkay 設定を取得する

説明

public bool MongoDB::getSlaveOkay ( void )

読み込みのスレーブへの分散についての情報は、このマニュアルの 問い合わせの節 を参照ください。

パラメータ

この関数にはパラメータはありません。

返り値

このインスタンスの slaveOkay の値を返します。
