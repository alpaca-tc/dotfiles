Imagick::drawImage

(PECL imagick 2.0.0)

Imagick::drawImage — 現在の画像上の ImagickDraw オブジェクトをレンダリングする

説明

bool Imagick::drawImage ( ImagickDraw $draw )

現在の画像上の ImagickDraw オブジェクトをレンダリングします。

パラメータ

draw

画像をレンダリングする描画操作。

返り値

成功した場合に TRUE を返します。
