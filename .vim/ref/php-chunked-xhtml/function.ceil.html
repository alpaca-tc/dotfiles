ceil

(PHP 4, PHP 5)

ceil — 端数の切り上げ

説明

float ceil ( float $value )

value の次に大きい整数値を返します。

パラメータ

value

丸める値。

返り値

value の次に大きい整数値を返します。 ceil() の返り値は float 型と なります。これは、float 値の範囲は通常 int よりも広いためです。

例

例1 ceil() の例

<?php
echo ceil(4.3);    // 5
echo ceil(9.999);  // 10
echo ceil(-3.14);  // -3
?>
