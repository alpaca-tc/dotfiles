mb_decode_mimeheader

(PHP 4 >= 4.0.6, PHP 5)

mb_decode_mimeheader — MIME ヘッダフィールドの文字列をデコードする

説明

string mb_decode_mimeheader ( string $str )

エンコードされた MIME ヘッダの文字列 str をデコードします。

パラメータ

str

デコードする文字列。

返り値

内部文字エンコーディングでデコードされた文字列を返します。
