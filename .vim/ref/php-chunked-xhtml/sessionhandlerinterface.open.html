SessionHandlerInterface::open

(PHP 5 >= 5.4.0)

SessionHandlerInterface::open — セッションを開始する

説明

abstract public bool SessionHandlerInterface::open ( string $save_path , string $session_id )

既存のセッションを再度初期化するか、あるいは新しいセッションを作成します。 セッションが開始したり session_start() が実行されたりしたときにコールされます。

パラメータ

save_path

セッションを格納/取得するパス。

session_id

セッションの id。

返り値

返り値 (通常は、TRUE が成功そして FALSE が失敗を表します)。この値は PHP で内部的に処理されるものであることに注意しましょう。
