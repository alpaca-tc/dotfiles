eio_ftruncate

(PECL eio >= 0.0.1dev)

eio_ftruncate — Truncate a file

説明

resource eio_ftruncate ( int $fd [, int $offset = 0 [, int $pri = EIO_PRI_DEFAULT [, callback $callback = NULL [, mixed $data = NULL ]]]] )

eio_ftruncate() causes a regular file referenced by fd file descriptor to be truncated to precisely length bytes.

パラメータ

fd

The file descriptor.

offset

Offset from beginning of the file

pri

リクエストの優先順位。EIO_PRI_DEFAULT、EIO_PRI_MIN、EIO_PRI_MAX あるいは NULL を指定します。 NULL を渡した場合、pri は内部的に EIO_PRI_DEFAULT となります。

callback

callback 関数が、リクエスト終了時にコールされます。 この関数のプロトタイプは、

void callback($data, $result);
に一致しなければなりません。 $data は、リクエストに渡すカスタムデータです。 $result にはリクエスト固有の結果が格納されます。基本的には、対応するシステムコールが返すそのままの値となります。

data

Arbitrary variable passed to callback.

返り値

eio_ftruncate() returns request resource on success or FALSE on error.
