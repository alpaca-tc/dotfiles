gc_enabled

(PHP 5 >= 5.3.0)

gc_enabled — 循環参照コレクタの状態を返す

説明

bool gc_enabled ( void )

循環参照コレクタの状態を返します。

パラメータ

この関数にはパラメータはありません。

返り値

ガベージコレクタが有効な場合に TRUE、そうでない場合に FALSE を返します。

例

例1 gc_enabled() の例

<?php
if(gc_enabled()) gc_collect_cycles();
?>
