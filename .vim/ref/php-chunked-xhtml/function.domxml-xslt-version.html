domxml_xslt_version

(PHP 4 >= 4.2.0)

domxml_xslt_version — XSLT ライブラリのバージョンを取得する

説明

int domxml_xslt_version ( void )

XSLT ライブラリのバージョンを取得します。

返り値

XSLT ライブラリのバージョン番号を整数で返します。

例

例1 domxml_xslt_version() の例

<?php

echo domxml_xslt_version();

?>

上の例の出力は、 たとえば以下のようになります。

10112
