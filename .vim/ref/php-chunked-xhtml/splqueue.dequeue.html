SplQueue::dequeue

(PHP 5 >= 5.3.0)

SplQueue::dequeue — キューからノードを取り出す

説明

mixed SplQueue::dequeue ( void )

value をキューの先頭から取り出します。

注意:

SplQueue::dequeue() は SplDoublyLinkedList::shift() のエイリアスです。

パラメータ

この関数にはパラメータはありません。

返り値

キューから取り出したノードの値を返します。
