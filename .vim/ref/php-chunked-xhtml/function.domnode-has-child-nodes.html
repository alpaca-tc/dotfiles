DomNode::has_child_nodes

(PHP 4 >= 4.1.0)

DomNode::has_child_nodes — ノードが子ノードを有しているかを調べる

説明

bool DomNode::has_child_nodes ( void )

この関数は、ノードが子ノードを有しているかを調べます。

domnode_child_nodes() も参照ください。
