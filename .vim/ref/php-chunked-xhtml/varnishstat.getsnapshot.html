VarnishStat::getSnapshot

(PECL varnish >= 0.3)

VarnishStat::getSnapshot — Get the current varnish instance statistics snapshot

説明

public array VarnishStat::getSnapshot ( void )

パラメータ

この関数にはパラメータはありません。

返り値
