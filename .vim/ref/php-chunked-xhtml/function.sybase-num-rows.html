sybase_num_rows

(PHP 4, PHP 5)

sybase_num_rows — 結果における行の数を取得する

説明

int sybase_num_rows ( resource $result )

sybase_num_rows() は、結果セットの行数を返します。

パラメータ

result

返り値

結果セットの行の数を返します。

参考

  • sybase_num_fields() - 結果におけるフィールドの数を取得する
  • sybase_query() - Sybase クエリを送信する
  • sybase_fetch_row() - 行を配列として取得する
