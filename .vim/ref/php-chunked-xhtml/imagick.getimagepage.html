Imagick::getImagePage

(PECL imagick 2.0.0)

Imagick::getImagePage — ページのジオメトリを返す

説明

array Imagick::getImagePage ( void )

画像のページジオメトリを配列で返します。配列のキーは "width"、"height"、"x" および "y" です。

返り値

画像のページジオメトリを配列で返します。配列のキーは "width"、"height"、"x" および "y" です。

エラー / 例外

エラー時に ImagickException をスローします。
