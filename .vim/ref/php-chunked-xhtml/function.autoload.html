__autoload

(バージョン情報なし。おそらく SVN 版にしか存在しないでしょう)

__autoload — 未定義のクラスのロードを試みる

説明

void __autoload ( string $class )

この関数を定義すれば、クラスのオートローディング を有効にできます。

パラメータ

class

ロードするクラスの名前。

返り値

値を返しません。

参考

  • spl_autoload_register() - 指定した関数を __autoload() の実装として登録する
