SplHeap::extract

(PHP 5 >= 5.3.0)

SplHeap::extract — ヒープの先頭からノードを取り出す

説明

mixed SplHeap::extract ( void )

パラメータ

この関数にはパラメータはありません。

返り値

取り出したノードの値を返します。

エラー / 例外

データが空のときは RuntimeException をスローします。
